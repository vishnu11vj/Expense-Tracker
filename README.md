# 💸 Expense Tracker

A personal finance tracker to log income and expenses.

## Features
- Add income and expense transactions with categories
- Live balance, total income, and total expense summary
- Category icons for visual clarity
- Spent vs earned progress bar
- Data saved in localStorage

## Built With
- HTML5
- CSS3
- Vanilla JavaScript
- localStorage API

## How to Run
Clone the repo and open `index.html` in your browser.
